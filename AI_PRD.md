# AI Page — PRD



## Objective

Present AI capabilities through structured, layered storytelling using alternating visual sections with supporting sub-content blocks.



---



## S1 — Hero Section



**Layout**

- Full viewport (100vh)

- Background image: `../../../assets/images/ai-page-cover.jpg`

- Dark overlay / gradient applied



**Position**

- Content positioned at Top ~60%, Left ~10%



**Content**

- Tagline (short, bold)

- Supporting subheadline (1–2 lines)



---



## S2 & S4 — Feature Block (Image Left + Content Stack)



**Background**

- `--color-secondary`



**Layout Structure**

- **Top Layer:** `[ Image ] [ Main Content ]`

- **Bottom Layer:** `[ Content 2 ] [ Content 3 ]`



**Details**

- **Top Left:** Large image (dominant visual)

- **Top Right:** Eyebrow (subtle highlight), Large headline (2–3 lines)

- **Bottom Row (2 columns):** Two supporting content blocks (Small title, Short description)



---



## S3 & S5 — Feature Block (Content Left + Image Right)



**Background**

- `--color-primary`



**Layout Structure**

- **Top Layer:** `[ Main Content ] [ Image ]`

- **Bottom Layer:** `[ Content 2 ] [ Content 3 ]`



**Details**

- **Top Left:** Eyebrow, Large headline, Supporting paragraph

- **Top Right:** Large image

- **Bottom Row:** Two supporting content blocks (same as S2)



---



## S6 — CTA Section

**Background**

- `--color-secondary`



**Layout**

- `[ Large Content ] [ Visual ]`



**Left**

- Large multi-line headline

- Supporting paragraph

- CTA button



**Right**

- Illustration / visual



---



## S7 — Form Section



**Background**

- `--color-tertiary`



**Layout**

- `[ Heading ] [ Form ]`



**Left (~40%)**

- Large heading (multi-line)



**Right (~60%)**

- Form fields: First Name, Last Name, Email, Company, Consent text

- Submit button



---



## Color Usage Strategy

- **`--color-secondary`**: Dark premium sections (Hero, visual impact)

- **`--color-primary`**: Light readable sections

- **`--color-tertiary`**: Form / conversion area

- **`--color-highlight`**: Very limited usage (Eyebrow, CTA only)



---



## Layout Pattern Summary

- **Alternating Blocks:** Image Left/Content Right followed by Content Left/Image Right.

- **Layered Structure:** Each block features a Top layer for the main message and a Bottom layer for supporting mini-content.



---



## Responsive Behavior

- **Desktop:** 2-column layout

- **Tablet:** Reduced spacing/gap

- **Mobile:**

  - Stack vertically

  - Order: Main Content → Image → Supporting blocks



---



## Design Principles

- Layered storytelling (Main + supporting content)

- Clean, structured, premium aesthetic

- Strong spacing rhythm (`py-20`, `gap-16`)

- Minimal accent usage

- No heavy animations



---



## Notes

- Maintain consistency across all sections

- Avoid clutter

- Prioritize visual hierarchy and clarity
